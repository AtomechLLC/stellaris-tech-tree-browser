| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Tampering/Elevation (XSS) — PRIMARY high-severity item | Node labels (tech.name) via Sigma label renderer; any React rendering of tech strings in chrome | mitigate | ALL game strings render as plain text: Sigma draws labels on canvas (no DOM/HTML), React chrome uses text nodes (auto-escaped). NEVER `dangerouslySetInnerHTML` (grep-gated to zero). Paradox §color§!/$var$ markup displays verbatim as text, never injected as HTML (D-05, SCHEMA.md). | T-02-05 | Information disclosure / spoofing (theme drift) | getComputedStyle bridge | accept | Reads same-origin CSS custom properties only; no external/user input crosses this boundary. Low risk. |
